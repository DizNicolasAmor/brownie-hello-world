# brownie-hello-world

This is a hello world app built with brownie.

## Requirements:

- brownie
- yarn
- ganache-cli

## Docs:

- https://github.com/eth-brownie/brownie
- https://classic.yarnpkg.com/en/
- https://www.npmjs.com/package/ganache-cli

## Setup

$ brownie compile

$ brownie run scripts/deploy_hello_world.py
